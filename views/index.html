Conceito

O AutoDesliga é uma aplicação simples, porém muito útil, que permite ao usuário programar o desligamento automático do computador de forma segura e conveniente.

É especialmente indicado para aqueles que assistem a vídeos ou ouvem música antes de dormir e não querem deixar o PC ligado a noite toda. Também é ideal para quem deseja economizar energia ou precisa de um desligamento temporizado para outros cenários, como downloads e backups.

Como Funciona

Nosso sistema utiliza os comandos nativos do Windows para agendar o desligamento. Basta informar um horário exato (HH:MM) ou um intervalo em minutos. É possível também cancelar a qualquer momento, se você mudar de ideia.

Internamente, o aplicativo envia o comando shutdown /s /t SEGUNDOS, que instrui o Windows a desligar de forma segura após o período especificado. O cancelamento é feito via shutdown /a, evitando o desligamento pendente.
