Conceito

O AutoDesliga é uma aplicação simples, porém muito útil, que permite ao usuário programar o desligamento automático do computador de forma segura e conveniente.

Ideal para quem assiste vídeos ou ouve música antes de dormir e não quer deixar o PC ligado toda a noite. Também perfeito para economizar energia e programar desligamentos após downloads ou backups.

Como Funciona

Nosso sistema utiliza os comandos nativos do Windows para agendar o desligamento de forma confiável. Basta informar um horário exato ou um intervalo em minutos, com a flexibilidade de cancelar quando necessário.

Programação

Utilize o formulário para configurar o tempo desejado para o desligamento automático.

Execução

O aplicativo converte seu input para o comando nativo do Windows.

# Comando usado para agendamento:

shutdown /s /t SEGUNDOS

# Comando usado para cancelamento:

shutdown /a

Configuração

Utilize o painel abaixo para escolher um horário específico ou um intervalo em minutos para desligar o computador. Você pode cancelar o agendamento a qualquer momento.

Agende o Desligamento

min
